# Documentation

This is the free-form documentation tree for Kaneer.
Use it for guides, tutorials, runbooks, and any reference material that does not need
the governance of the [Docs CMS](../docs-cms/README.md).

Add new pages as Markdown files under `docs/`. Order them with `sidebar_position`
frontmatter, and label subfolders with a `_category_.json` file.
